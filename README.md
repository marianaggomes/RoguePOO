# RoguePOO

## Descrição
RoguePOO é um jogo desenvolvido em Java para a UC de Programação Orientada para Objetos, que utiliza uma arquitetura modular composta por dois projetos:

- **GraphPack**: biblioteca responsável por funcionalidades gráficas e utilitários do jogo.  
- **RoguePOO**: projeto principal que implementa a lógica do jogo e depende do GraphPack.

O projeto foi desenvolvido no Eclipse e agora pode ser executado no **VS Code** através de um workspace que inclui ambos os módulos.

---

## Como Executar

1. **Abrir o Workspace no VS Code**
   - `File → Open Workspace… → ProjetoPOO.code-workspace`

2. **Compilar GraphPack**
   - O VS Code deve compilar automaticamente.
   - Caso necessário: `CTRL + SHIFT + B → Java: Compile Project`

3. **Executar RoguePOO**
   - Abre a classe com `public static void main(String[] args)`  
   - Clica no botão **Run** acima do método `main`.

> **Nota:** É importante abrir sempre o workspace, não apenas a pasta principal, para que o VS Code reconheça os dois projetos corretamente.
